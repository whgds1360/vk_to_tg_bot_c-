# VK → Telegram Bot (C#)

Простой бот для пересылки сообщений из **ВКонтакте** (группа/сообщество) в **Telegram**.

### О проекте

Этот бот предназначен для мониторинга новых сообщений в группе ВКонтакте и их автоматической пересылки в указанный Telegram-чат. 

**Технологии:**
- C# / .NET
- [VkNet](https://github.com/vknet/vk) — работа с VK API
- [Telegram.Bot](https://github.com/TelegramBots/Telegram.Bot) — Telegram Bot API
- `VkNetLongpoll` (попытка использования)

---

### Статус проекта

**❌ Проект не завершён**

К сожалению, довести бота до рабочего состояния не удалось.

#### Причина остановки разработки

В последнем коммите я написал:

> "Засел на LongPoll. Ни одна библиотека не хочет нормально работать:  
> либо методы отсутствуют, либо они устаревшие, либо LongPoll ловит только команды, а обычные сообщения от пользователей — нет."

Несмотря на множество попыток (ручной Long Poll, `VkNet.Extensions.Polling`, `VkNetLongpoll` и т.д.), стабильного получения всех входящих сообщений от пользователей в группе добиться не получилось.

### Что сделано

- Структура проекта и разделение ответственности
- Инициализация VkApi и Telegram бота
- ResourceManager для хранения токенов

### Что не работает

- Надёжное получение всех сообщений через Long Poll из группы ВКонтакте

### Как запустить (если вдруг заработает)

Перед запуском создайте файл `.env` в корне проекта:

```env
# Telegram
#TOKEN от BotFather
TG_BOT_TOKEN="Ваш токен"
#Список ID бесед вк, которые слушаем
LIST_OF_LISTEN=1111111111,1111111111
#VK TOKEN
VK_TOKEN="Токен вк"
#ID сообщества в вк которому привязан бот
VK_COMMUNITY_TOKEN=111111111
#ID чата в тг с вашим ботом
CHAT_ID=1111111111
```
